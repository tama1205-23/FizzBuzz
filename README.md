# FizzBuzz
<p>
  まず、FizzBuzz問題についてGoogleで検索し、概要や実装例を調べました。<br>
複数のサイトを確認する中で、FizzBuzz問題にはさまざまな書き方があり、人によってコードの記述方法が異なることを知りました。<br>
<br>
その後、参考サイトのコードをVisual Studio Codeにて実際に手で入力していきました。<br>
参考サイト：https://sitest.jp/blog/?p=3822<br>
<br>
入力した「for(ver i=1; i<=100; i++)」の部分でエラーが発生していたため、自分なりに原因を考え、スペースの有無が原因なのではないかと考えて、「for(veri=1; i<=100; i++)」と、入力して確認を行いました。<br>
しかし、エラー表示は消えたものの、実際にChromeのConsoleで動作確認を行なった際には、正常に動作しませんでした。<br>
<br>
その後、エラー表示が出ていたコードに何か問題があるのではと考え、スペースを追加したり、削除したりをしながら確認を行いましたが、改善しませんでした。<br>
そこで、ChatGPTにコードを確認してもらった結果、「var」を誤って「ver」と入力していたことが原因であると判明しました。<br>
誤っていたコードを修正し、再度GoogleのConsoleにて動作確認を行なったところ、１〜100までのFizzBuzzが正常に表示されました。<br>
<br>
その後、課題条件に合わせるため、for文内の「i<=100」が最大数を指定しているのではと考え、「i<=20」に修正を行いました。<br>
また、参考にしたサイト内では、3と5の倍数の場合の表示が「Fizz,Buzz」となっていたため、カンマを削除し、「FizzBuzz」に修正しました。<br>
<br>
今回の課題では、参考サイトをもとにコードを作成しましたが、ただ写すだけではなく、「このコードは何を指定しているのか」「どの部分が動作に関係しているのか」を自分なりに考えながら進めました。<br>
実際にエラーの原因を調べて修正することで、コードの意味や動作への理解を深めながら取り組むことができました。<br>
</p>
